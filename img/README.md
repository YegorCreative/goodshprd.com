# Good Shepherd Icon Integration

## Required Image File

Save your shepherd dog icon as:
**`shepherd-icon.png`**

### Specifications:
- **Recommended size:** 512x512px (will be scaled down automatically)
- **Format:** PNG with transparent background
- **Colors:** The tan/beige tones with black outlines match the brand palette
- **Usage:** 
  - Header logo (32x32px display)
  - Browser favicon (16x16, 32x32)
  - Apple touch icon (180x180)

### Where It's Used:
- ✓ All page headers next to "Good Shepherd" text
- ✓ Browser tab favicon
- ✓ Apple touch icon (when saved to home screen)

### Current Status:
The HTML and CSS are ready. Just add `shepherd-icon.png` to this `/img/` folder.

### Alternate Formats (Optional):
You can also create:
- `favicon.ico` (for older browsers)
- `shepherd-icon.svg` (for scalable vector version)

The site will work with just the PNG file.
